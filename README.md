# OpenPolicy

Policy-as-code platform for busy founders

# Notes

A toolkit for generating legal policies from code. Define the policies as type-safe objects and then generate usable markdown/html/pdfs from those files.

- SDK - A toolkit for generating legal policies from code. Define the policies as type-safe objects and then generate usable markdown/html/pdfs from those files.
- Vite Plugin - A Vite plugin for generating legal policies from code. Define the policies as type-safe objects and then generate usable markdown/html/pdfs from those files.
- CLI - A CLI for generating legal policies from code. Define the policies as type-safe objects and then generate usable markdown/html/pdfs from those files.

## Ideas
- PR Bot - Checks for changes to the code which impact policy.
- Webhooks - Trigger email notifications when policies change.
- Prompt/Skill - Used to handle the initial setup.
